# PokerSettler 🃏

אפליקציית ווב mobile-first לניהול משחקי פוקר ביתיים וחישוב התחשבנות אוטומטי.

## תכונות עיקריות

- **לובי**: יצירת משחק חדש או הצטרפות למשחק קיים באמצעות קוד חדר
- **ניהול שחקנים**: הוספת buy-ins מרובים ו-cash-out לכל שחקן
- **וולידציה**: בדיקה שסכום ה-buy-ins שווה לסכום ה-cash-outs
- **אלגוריתם התחשבנות**: חישוב מינימלי של העברות כסף בין השחקנים
- **עיצוב mobile-first**: מותאם לטלפונים חכמים עם מצב כהה וירוק פוקר

## התקנה והפעלה

```bash
# התקנת תלויות
npm install

# הפעלת שרת הפיתוח
npm start
```

האפליקציה תיפתח בכתובת: http://localhost:3000

## מבנה הפרויקט

```
src/
├── components/
│   ├── Lobby.js          # מסך הכניסה והצטרפות
│   └── GameRoom.js       # מסך המשחק הראשי

│   └── settlementLogic.js # אלגוריתם חישוב ההתחשבנות
├── App.js                # קומפוננט ראשי
├── index.js              # נקודת כניסה
└── index.css             # עיצוב Tailwind
```

## אלגוריתם ההתחשבנות

האלגוריתם מחשב:
1. **נטו לכל שחקן**: הפרש בין cash-out ל-buy-ins
2. **הפרדה לזוכים ומפסידים**
3. **מינימיזציה של העברות**: אלגוריתם greedy שמתאים זוכים למפסידים
4. **וולידציה**: בדיקה שסכום כל הרווחים והפסדים = 0

### דוגמה:
- שחקן א': buy-in ₪100, cash-out ₪150 → רווח ₪50
- שחקן ב': buy-in ₪100, cash-out ₪80 → הפסד ₪20  
- שחקן ג': buy-in ₪100, cash-out ₪70 → הפסד ₪30

**תוצאה**: שחקן ב' משלם ₪20 לשחקן א', שחקן ג' משלם ₪30 לשחקן א'

## טכנולוגיות

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **State Management**: React Hooks + localStorage
- **Mobile-First**: Responsive design

## פיתוח עתידי

- **Backend**: Node.js + Socket.io לסנכרון בזמן אמת
- **Database**: שמירת היסטוריית משחקים
- **PWA**: התקנה כאפליקציה על הטלפון
- **סטטיסטיקות**: מעקב אחר ביצועי שחקנים לאורך זמן
